## Instructions
Suppose you are an architect and you need to place different objects in a 10*10 room. The type of the object is given in the format of type: number, size (length*width), assuming that each object is a rectangle. For each task, give a layout different from the predefined layout.

- Place all items in one room
- All items must be placed, and the quantity and variety cannot be reduced

- Different objects in the generated layout cannot overlap.
